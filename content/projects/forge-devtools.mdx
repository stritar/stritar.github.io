## Overview

Developers judge tools in seconds. Forge's onboarding took twenty minutes and
a support ticket. Lorem ipsum dolor sit amet, consectetur adipiscing elit.

> The best developer experience is the one you never have to think about.

## The challenge

- Setup required reading three docs before the first success.
- Defaults were missing, so everyone configured everything.
- Errors explained *what* broke but never *what to do next*.

## Approach

We made the first command do something useful with zero flags, then layered
power behind opt-in. Error messages now end with a suggested next step.

<Figure alt="Forge CLI first-run experience" caption="One command, sensible defaults, an obvious next step." />

## Outcomes

- **90 seconds** median time-to-first-success, down from ~20 minutes.
- **−70%** setup-related support tickets.
- A quickstart that fits on one screen.
